# TRIPNEXUS Frontend

React + TypeScript + Tailwind CSS frontend application for the Travel Agency Lead Management System.

## Features

- Lead management dashboard
- Real-time notifications
- Mobile-responsive design with dark mode
- PDF generation and sharing
- Interactive lead cards and pipeline
- User authentication with JWT

## Setup

```bash
npm install
npm run dev
```

Frontend runs on http://localhost:3000

## Build

```bash
npm run build
```

## Project Structure

- `src/components/` - Reusable React components
- `src/pages/` - Page components
- `src/context/` - Auth and state management
- `src/types/` - TypeScript type definitions
- `src/utils/` - Helper functions and API services
- `public/` - Static assets
